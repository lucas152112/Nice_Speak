## 分數計算

### 評分項目

| 項目 | 分數 | 說明 |
|------|------|------|
| **發音** | 30 分 | 語音清晰度、發音準確性 |
| **語法** | 30 分 | 句子結構、時態正確性 |
| **用詞** | 20 分 | 詞彙選擇、專業術語 |
| **流暢度** | 20 分 | 說話節奏、停頓自然度 |
| **總計** | **100 分** | |

### 分數計算公式

```
總分 = 發音分 + 語法分 + 用詞分 + 流暢度分

發音分 (0-30) = AI 評估發音準確度 × 0.3
語法分 (0-30) = AI 評估語法正確度 × 0.3
用詞分 (0-20) = AI 評估詞彙適切度 × 0.2
流暢度分 (0-20) = AI 評估流暢度 × 0.2
```

## 升級條件

### 連續達標

```
連續 3 次練習平均分數達到該等級門檻，即可升級。

範例（等級 0 → 1）：
- 等級 0 門檻：10 分
- 練習 1：12 分 ✅
- 練習 2：15 分 ✅
- 練習 3：11 分 ✅
─────────────────
升級成功！ → 等級 1
```

### 累計達標

```
累計 6 次練習平均分數達到該等級門檻，也可升級。

範例（等級 0 → 1）：
- 等級 0 門檻：10 分
- 練習 1：5 分 ❌
- 練習 2：8 分 ❌
- 練習 3：12 分 ✅
- 練習 4：7 分 ❌
- 練習 5：11 分 ✅
- 練習 6：9 分 ❌ (累計 4 達標 / 6)
- 練習 7：10 分 ✅ (累計 5 達標 / 6)
- 練習 8：12 分 ✅
─────────────────
升級成功！ → 等級 1
```

## 常見問題

### Q1: 降級嗎？

A: 不會。等級只升不降，保護學習信心。

### Q2: 連續和累計可以混合嗎？

A: 可以。系統會自動計算最有利的組合。

### Q3: 練習中斷怎麼辦？

A: 連續中斷不影響累計，隨時可以繼續。

### Q4: 等級 5 和 10 的折扣可以疊加嗎？

A: 不行，只能選擇較高折扣（60%）。
